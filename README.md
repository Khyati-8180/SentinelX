# 🛡️ SentinelX

## Live Demo

https://sentinelx-dashboard.streamlit.app/ 

## Overview

SentinelX is an AI-powered Insider Threat Detection Platform designed to help organizations monitor employee activities, detect suspicious behavior, and improve cybersecurity awareness through intelligent dashboards and analytics.

The platform analyzes employee login records, device activity, and web usage to identify potential insider threats and provide actionable security insights.
